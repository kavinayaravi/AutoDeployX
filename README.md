# AutoDeployX 🚀

AutoDeployX is a real-time DevOps pipeline project that automates the build, test, and deployment of a Node.js application using popular DevOps tools like Git, GitHub, Jenkins, Docker, Kubernetes, Prometheus, and Grafana — all deployed on cloud infrastructure.

## 💡 Project Highlights
- Node.js web app built with Express
- CI/CD pipeline set up with Jenkins
- Dockerized app and pushed to DockerHub
- Kubernetes cluster deployment (minikube or cloud-based)
- Application monitoring with Prometheus + Grafana
- All code backed up on GitHub for version control

## 📦 Tech Stack
- Node.js + Express
- Git & GitHub
- Jenkins
- Docker
- Kubernetes
- Prometheus
- Grafana
- AWS EC2 (or any VM/cloud platform)
